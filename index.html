Web User Clustering

User data exploration and pattern discovery

My project is about user clustering. Millions of people in Hong Kong browse websites everyday. Can we find any user browsing pattern? Can we cluster the users according to their browsing history? These are the questions I want to answer before I start this project. In order to see whether there is a pattern we need the user data. Where can we get the user browsing data? The answer is ad network, which is run by my company.

Ad Network

Ad network serves ads from advertisers on publishers' websites

When you visit any site usually ads will be served. When you see the ad, the ad server also get some of your browser information. For exmaple, [show sample sites]

User Selection

Active users

  1. More than 20 views a day
  2. Appear in the network more than 20 days in 30-day range (Feb. 15 ~ Mar. 16)
  3. User from Hong Kong
There are millions of internet users in Hong Kong, I can't collect all of them. So I set some criterias for user selection. I want only the "active users". browse more than 20 pages a day; appear in our network more than 20 days in the 30-day range. The data I've collected are from Feb. 15 to Mar. 16.; And I only care about the users from Hong Kong
